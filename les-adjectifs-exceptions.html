Les adjectifs – Besonderheiten

Erklärungen & Übungen zu den wichtigsten Ausnahmen

1 Adjektive vor dem Nomen

Die meisten Adjektive stehen im Französischen hinter dem Nomen – das ist die Grundregel. Aber eine kleine Gruppe sehr häufiger, kurzer Adjektive stellt sich davor. Du kannst sie mit dem Merksatz BAGS einteilen:

✨ Schönheit (beauty)
beau, joli
📅 Alter (age)
jeune, vieux, nouveau
👍 Gut/Schlecht (good)
bon, mauvais
🏔 Größe (size)
grand, petit

Beispiel: C'est un bon film. – Das ist ein guter Film. (bon steht vor film)

2 Die „großen Drei": beau, nouveau, vieux

Diese drei Adjektive verändern ihre Form besonders stark. Steht das männliche Nomen mit einem Vokal (a, e, i, o, u) oder einem stummen h, braucht man eine „weiche Brücke", damit zwei Vokale nicht aufeinandertreffen. Dann nimmt das Adjektiv eine Sonderform an:

Adjektiv Mask. Sing.
(vor Konsonant)		 Mask. Sing.
(vor Vokal / stumm. h)		 Fem. Sing. Mask. Plur. Fem. Plur.
schön beau bel belle beaux belles
neu nouveau nouvel nouvelle nouveaux nouvelles
alt vieux vieil vieille vieux vieilles

Beispiele: un beau jardin – aber: un bel arbre (arbre beginnt mit Vokal)

3 Besondere Endungen – weibliche Form

Die Grundregel lautet: Männliche Grundform + -e = weibliche Form. Manche Adjektive folgen aber einem anderen Muster:

-e → -e (gleich)
rouge → rouge
männl. = weibl.
-eux → -euse
heureux → heureuse
-if → -ive
sportif → sportive
-er → -ère
premier → première
-on/-en/-el → Konsonant doppeln + -e
bon → bonne
mignon → mignonne

4 Besondere Pluralformen

Grundregel: Singular + -s = Plural. Ausnahmen:

-s / -x → unverändert
gris → gris
faux → faux
-eau → -eaux
beau → beaux
nouveau → nouveaux
-al → -aux
génial → géniaux
local → locaux
unveränderlich
marron → marron
orange → orange
